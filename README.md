# Ventana Nails & Spa — Richmond, TX

Website for **Ventana Nails & Spa**, a premium nail salon located in Richmond, TX.

## About

Single-page website built with plain HTML, CSS, and JavaScript (no frameworks).

**Business info:**
- Address: 9711 Mason Rd Ste 120, Richmond, TX 77407
- Phone: (832) 422-6979
- Hours: Mon–Fri 9:30 AM–7 PM · Sat 9 AM–7 PM · Sun 11 AM–5:30 PM
- Rating: ⭐ 4.8 stars · 667+ Google Reviews

## Features

- Black, white & gold luxury aesthetic
- Cormorant Garamond + Montserrat typography
- Glassmorphism nav, scroll fade-in animations
- Services section with 5 tabs (Manicures, Pedicures, Nails, Waxing, Drinks Menu)
- Expandable service descriptions
- Reviews carousel (drag to scroll)
- Photo gallery grid
- Google Maps embed
- Language selector overlay (English / Español / Tiếng Việt)
- Mobile responsive with hamburger menu
- Floating Book Now button on mobile
